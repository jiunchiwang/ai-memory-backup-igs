## 重點摘要

**是什麼：** 一套讓 AI agent（Claude Code、Cursor、Kiro、Codex 等 15+ 平台）在產 UI 時自動套用專業設計規則的 skill 包。

**核心能力：**
- **161 條行業推理規則** — 根據產品類型（SaaS、金融、醫療、電商等）自動推薦設計系統
- **67 種 UI 風格** — Glassmorphism、Brutalism、Neumorphism、Bento Grid、AI-Native UI 等
- **161 色盤** — 與產業一對一對應
- **57 組字體配對** — 附 Google Fonts import
- **25 種圖表推薦**（dashboard 用）
- **99 條 UX guideline**

**運作方式：** 你跟 AI 說「幫我做一個 SaaS landing page」，skill 會自動觸發五步流程：
1. 多域搜尋（161 產業類別 + 67 風格 + 色盤 + 字型 + landing page pattern）
2. BM25 排序推理引擎
3. 產出完整設計系統（pattern + style + colors + typography + effects + anti-patterns）
4. 依此生成程式碼
5. 交付前檢查（對比度、hover state、responsive 等）

**支援的技術棧：** React / Next.js / Vue / Nuxt / Svelte / Astro / Flutter / SwiftUI / React Native / HTML+Tailwind / shadcn/ui / Angular / Laravel 等 15 種。

**安裝方式（Kiro）：**
```bash
npm install -g uipro-cli
cd /path/to/your/project
uipro init --ai kiro
```

**v2.0 旗艦功能：** Design System Generator — 分析需求後一次產出完整設計系統（含反模式警告和交付檢查清單），可 persist 到 `design-system/MASTER.md` 跨 session 使用。
