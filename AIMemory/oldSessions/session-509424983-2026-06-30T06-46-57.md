---

### 🤖 Assistant (下午2:34:50)

確認方式：

**在 claude.ai 上：**
1. 點左下角你的名字/頭像
2. 你的角色和方案類型會顯示在組織名稱旁邊
3. 或進 **Organization settings > Members**，每個 seat 旁會標 `Standard` 或 `Premium`

**在 Claude Code CLI 裡：**
- 跑 `/status` — 會顯示你的帳號類型和可用 model
- 跑 `/model` — Premium 才能看到 Claude Code 專用的模型選項

**最簡單的判斷法：**
- 能用 Claude Code → **Premium**
- 不能用 Claude Code（只有 claude.ai 網頁）→ **Standard**

**如果你是 Admin：**
- Organization settings > Members 頁面可以看到每個成員的 seat tier，也可以在那裡把 Standard 升級為 Premium（不用重買 seat）。
