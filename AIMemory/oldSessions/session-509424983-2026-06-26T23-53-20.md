## SkillsMP（skillsmp.com）概覽

**本質**：一個 SKILL.md 的公開目錄/市集，收錄了 ~196 萬份從 GitHub 公開 repo 爬來的 skill 檔案。

**格式相容性**：它索引的就是標準 `SKILL.md` 格式——跟你目前用的完全一樣（Claude Code / Codex / Kiro 通用）。所以上面的 skill 拿下來直接放進你的 skill 目錄就能用。

**主要功能**：
- **搜尋**：按關鍵字、職業分類（867 個 SOC 分類）、或 category（Tools / Development / Data & AI 等）瀏覽
- **來源透明**：每個 skill 都連結回 GitHub 原始 repo，可以看維護狀態
- **REST API**：`GET /api/v1/skills/search?q=...`，匿名 50 次/天，有 key 500 次/天
- **熱門 skill**：anthropics 官方的 frontend-design / skill-creator / docx，你已有的 ui-ux-pro-max，以及 browser-use、code-reviewer 等

**跟你的 bridge 的關係**：
- SkillsMP 只是「發現」管道，不提供安裝機制
- 找到想用的 → 去 GitHub 抓 SKILL.md（可能還有 `references/`、`assets/` 子目錄）→ 放進正本目錄
- Bridge 的 embedding router 會自動把新 skill 的 description 向量化納入 routing

**實用場景**：
- 想找某領域（如 code review、API design、testing）的現成 best practice
- 借鏡別人怎麼寫 skill（結構、觸發條件、references 運用）
- 用 API 批量搜尋再挑選

## `/skillsearch` 構想

### 使用方式

```
/skillsearch <關鍵字>
```

例：`/skillsearch code review`、`/skillsearch react performance`

### 功能流程

1. **使用者輸入關鍵字** → bridge 呼叫 SkillsMP API（`GET /api/v1/skills/search?q=...&limit=5&sortBy=stars`）
2. **列出結果** → 顯示 top 5 skill（名稱、作者/repo、description 前 80 字、stars）
3. **ASK 按鈕** → 每個結果附 inline keyboard 選項，點選後：
   - **「查看詳情」**→ 抓 GitHub 原始 SKILL.md 內容顯示
   - **「安裝」**→ 下載 SKILL.md 到正本目錄 `G:\AI\AI-canonical\skills\general\<name>\SKILL.md`，跑 `sync.ps1 -Apply` 投影

### 互動範例

```
使用者：/skillsearch typescript testing

Bot：
🔍 SkillsMP 搜尋結果：typescript testing

1. vitest-testing（anthropics/skills）⭐154k
   Unit and integration testing with Vitest...
2. playwright-e2e（anthropics/skills）⭐154k
   End-to-end testing for web applications...
3. jest-patterns（someuser/repo）⭐12k
   Jest testing patterns and best practices...

<<ASK:ss1|detail_1=查看 #1|detail_2=查看 #2|detail_3=查看 #3|install_1=安裝 #1|install_2=安裝 #2>>
```

### 技術面考量

| 項目 | 說明 |
|------|------|
| API 限制 | 匿名 50 次/天、10 次/分鐘（夠用，一天搜不了 50 次 skill） |
| API Key | 可選，有 key 升到 500 次/天 |
| 安裝路徑 | 依 `skill-workflow.md` 走正本 → sync 投影 |
| 子資源 | 若 skill 有 `references/`、`assets/` 子目錄需要額外 clone |
| 衝突檢測 | 安裝前檢查同名 skill 是否已存在 |
| 無 agent 介入 | 純 bridge 指令，不需 ACP session |

### 進階選項（可以之後加）

- `/skillsearch --category devops <關鍵字>` — 按分類篩
- `/skillsearch --install <github-url>` — 直接給 URL 安裝
- 安裝後自動 restart 讓 embedding router 納入新 skill
