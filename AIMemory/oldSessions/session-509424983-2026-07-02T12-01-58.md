2. **漸進生長** — 不需一次蓋 7 層，從第 1 層（一份 CLAUDE.md）開始，痛點出現再往上長。
3. **跨平台可攜** — 技能遵循開放標準 [agentskills.io](https://agentskills.io)，Claude Code / Codex / Antigravity 三家都支援。
4. **寫程式 agent 額外需要** — Verification（驗證才算完成）+ Scope（一次只做一件事）。
