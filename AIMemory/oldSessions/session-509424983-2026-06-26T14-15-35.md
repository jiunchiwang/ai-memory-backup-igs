### 二、手續費計算說明

台灣股票法定手續費上限為**成交金額 × 0.1425%**（買＋賣各收一次），券商給折扣是在這基礎上打折：

- 1.68 折 → 實際 = 0.1425% × 0.168 = **0.02394%**（永豐大戶投）
- 2.8 折 → 實際 = 0.1425% × 0.28 = **0.0399%**
- 6 折 → 實際 = 0.1425% × 0.6 = **0.0855%**

另外賣出時政府固定收**證交稅 0.3%**（ETF 為 0.1%），這跟券商無關。

### 三、依需求推薦

**🔥 小資存股族（台股定期定額為主）**
→ **國泰證券** 或 **永豐大戶投**
- 100 元起投、標的最多
- 國泰定期定額手續費 1 元起，永豐台股 1.68 折

**🌐 想買美股（複委託）**
→ **永豐大戶投** 或 **中信證券**
- 美股手續費 0.15%、低消 3 美元（業界最低區間）
- 國泰美股 ETF 有均一價方案也值得考慮

**📱 看重 APP 使用體驗**
→ **永豐大戶投**
- 業界公認 UI/UX 最佳
- 支援 Python API 程式交易

**🏢 重視據點（需要臨櫃服務）**
→ **元大證券**
- 全台據點最多（~130 家）
- 但手續費折扣相對保守

**💳 想整合日常銀行帳戶**
→ 選跟你主要銀行同集團的券商（少一張卡、自動扣款方便）

### 四、開戶前確認事項

1. **雙證件**：身分證 + 第二證件（健保卡/駕照）
2. **交割銀行帳戶**：多數券商要求同集團銀行，若沒有需額外開
3. **年齡**：滿 20 歲（2023 年起 18 歲可獨立開戶，但部分券商仍要求 20）
4. **一人可開多家**：不限一家券商，可同時開多家比較

### 五、注意事項

- 手續費折扣「可議」的券商（元大、國泰、中信）通常看你的交易量，開戶時可跟營業員談
- 定期定額手續費通常另有優惠（1元/筆或免手續費），但可能限期活動
- 複委託 vs 海外券商（如 IB、Firstrade）：複委託優勢是**台灣法律保障 + 報稅方便 + 遺產處理簡單**，劣勢是手續費較高
