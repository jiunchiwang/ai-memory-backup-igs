## 概覽

一個開源的 **語音互動 AI 伴侶**專案，搭配 **Live2D** 虛擬角色，支援即時語音對話、視覺感知（攝影機/螢幕截圖）、可完全離線運行。⭐ 10.6k stars、1.2k forks，Python 為主（96.6%）。

**命名緣由**：最初目標是用開源方案復刻閉源 AI VTuber「neuro-sama」。

---

## 核心功能

| 類別 | 重點 |
|------|------|
| **語音對話** | 即時語音交互、支援語音打斷（不需耳機，AI 不會聽到自己的聲音） |
| **視覺感知** | 攝影機、螢幕錄製、截圖——AI 能「看到」你和你的畫面 |
| **Live2D 角色** | 表情映射、觸控回饋（點擊/拖曳互動） |
| **桌面寵物模式** | 透明背景、全域置頂、滑鼠穿透 |
| **TTS 翻譯** | 可用中文聊天但 AI 說日文等 |
| **AI 主動說話** | 不需使用者先開口 |
| **聊天紀錄持久化** | 隨時切換回之前的對話 |

---

## 技術架構 & 支援後端

| 模組 | 支援選項 |
|------|---------|
| **LLM** | Ollama、OpenAI（含相容 API）、Gemini、Claude、Mistral、DeepSeek、智譜、GGUF、LM Studio、vLLM… |
| **ASR（語音辨識）** | sherpa-onnx、FunASR、Faster-Whisper、Whisper.cpp、Groq Whisper、Azure ASR… |
| **TTS（語音合成）** | sherpa-onnx、pyttsx3、MeloTTS、Coqui-TTS、GPTSoVITS、Bark、CosyVoice、Edge TTS、Fish Audio、Azure TTS… |
| **Agent 擴展** | 可繼承 Agent interface 接 HumeAI EVI、OpenAI Her、Mem0 等 |

---

## 平台 & 部署

- **跨平台**：Windows / macOS / Linux
- **GPU**：NVIDIA、非 NVIDIA、CPU、macOS GPU 加速（部分）
- **兩種前端**：Web 版 + Desktop Client（Electron 風格的桌面寵物）
- **Docker** 支援
- **包管理**：`uv`（Python）、`pixi.lock`
- **更新機制**：`uv run update.py`

## 簡單講差異核心

1. **Open-LLM-VTuber 強在「沉浸感」**——語音即時互動 + 視覺感知 + Live2D 表情，目標是復刻 neuro-sama 那種「直播型 AI VTuber / 陪伴體驗」。

2. **你的系統強在「能力與記憶」**——ACP 讓 agent 有完整工具使用能力（讀寫檔案、跑命令、MCP 外掛），加上自建的長期記憶 + topic 分類 + embedding recall + specialist 並行，是一個**能幹活的 AI 助手**，pet 只是其中一個展示層。

3. **互補可能性**：如果你的 desktop pet 想加語音互動 + 視覺感知，Open-LLM-VTuber 的 ASR/TTS pipeline 設計可以參考；反過來，他們缺的長期記憶和 tool-use 能力正好是你已經做好的。
