## SlotReels 旋轉機制

### 整體架構

SlotReels 採用**時間驅動 + 位移映射**的方式模擬轉輪滾動，不是逐幀移動節點，而是每幀根據「已經過的時間」算出「應該在哪個位置」。

### 核心資料結構

```
m_allColumns[plate][col] → ColumnSymbol
```

- **plateCount = 3**（Standard 模式）：下層 [0]、中層 [1]、上層 [2]，三條輪帶首尾相接形成「無限滾動帶」的視覺
- **plateCount = 1**（DropEntry 模式）：只有中層 [0]

### 旋轉生命週期

```
StartSpin() → [每幀] Spin(dt) → SpinAll() → Spin_RealSpinMultiple() → [收到結果] SetPlateInfo() → SetReelTimes() → [時間到] 停輪
```

#### 1. StartSpin()
- 重置時間計數器 `m_time = 0`
- 清空停輪時間 `m_reelStopTime[]`
- 預算各列轉一圈時間 `m_cachedSpinTimes[]`
- 重置符號切換狀態

#### 2. Spin(dt) — 每幀呼叫
```typescript
m_time += dt;
if (StopTime === 0 || m_time <= StopTime) → 繼續轉
else → m_isStop = true（全部停完）
```

#### 3. SpinAll() — 驅動所有列
對每個未鎖定的列，算出 `moveLength`：

```typescript
moveLength = ((curTime - riseTime) % spinTime / spinTime) * totalLength
```

這是一個**取餘循環**：時間對「轉一圈時間」取模，映射到 0 ~ totalLength 的位移。

#### 4. Spin_RealSpinMultiple() — 單列單盤面的位置計算

五個時間階段（依序判斷）：

| 階段 | 條件 | 行為 |
|------|------|------|
| **蓄力抬起** | `curTime < riseTimeCol` | 不動 |
| **抬起中** | `curTime < riseTimeCol + riseTime` | 向上移動 `upLength` |
| **等待其他列抬起** | `curTime < totalRiseTime` | 停在抬起位置 |
| **正常滾動** | `reelStopTime == null` 或 `curTime < stopTime + riseTime` | 用 `moveLength` 循環滾動 |
| **過頭下沉** | `curTime < stop + buf1` | 向下多滑 `downLength`（15px） |
| **回彈** | `curTime < stop + buf1 + buf2` | 從下沉位置彈回原位 |
| **最終定位** | else | 設定到精確停輪位置 |

方向由 `m_reelDirection`（Down=-1 / Up=1）控制。

### 符號切換機制（CheckMoveLength）

輪帶是**有限長度循環**，不是真的無限長。當 `moveLength > comparePos` 時：

1. 觸發 `ChangeSymbol()` — 把這條輪帶的符號換成新的一組
2. 回傳 `moveLength - totalLength` — 位置回到起點（視覺上無縫）

等於每轉一圈就換一次符號圖案，造成「無限轉輪帶」的錯覺。

### 停輪時間計算（SetReelTimes）

收到 server 結果後呼叫：

```typescript
reelStopTime[col] = spinTime × (timesOfSpin + baseSpin) + spinIntervalTimes × spinTime × (col - lockCount)
```

- `timesOfSpin`：已轉圈數（確保至少轉完當前圈）
- `m_timesOfBaseSpin = 2`：基礎最少轉 2 圈
- `m_spinIntervalTimes = 1`：每列多轉 1 圈（造成「由左到右依序停輪」效果）
- `lockCount`：鎖定列不計入間隔

最終 `StopTime = 最後一列停輪時間 + bufferTime01(0.1s) + bufferTime02(0.1s)`

### 快停（ClickStopBtn）

玩家按停止鈕：
- 重算每列的 `reelStopTime` 為「當前已轉圈數 + 1 圈」
- 所有列幾乎同時停（不再有列間隔）
- 使用 `tempoSetting` 或 fallback `(timeOfSpine + 2) × spinTime`

### 符號替換策略

| 時機 | 來源 |
|------|------|
| 滾動中（假輪帶） | `RandomSymbol()` 隨機生成 |
| 準備停輪（`reelPreTime` 後） | `m_plateInfoWithFakeSymbol`（server 結果 + 假符號補齊） |
| 最終停輪 | `CheckChangeToFinalSymbol()` 確保只設一次 |

### 關鍵參數

| 參數 | 預設值 | 說明 |
|------|--------|------|
| `m_spinTime` | 0.4s | 轉一圈時間 |
| `m_timesOfBaseSpin` | 2 | 最少轉幾圈 |
| `m_spinIntervalTimes` | 1 | 列間多轉幾圈 |
| `m_spinBufferTime01` | 0.1s | 過頭時間 |
| `m_spinBufferTime02` | 0.1s | 回彈時間 |
| `m_downLength` | 15px | 過頭下沉距離 |
| `m_riseTime` | 0s | 抬起時間（目前關閉） |

### 與 FillStrategy 的關係

- **Standard 模式**：走上述完整旋轉流程
- **Cascade/Tumble 模式**：`SpinState` 偵測到有 `EntryStrategy` 時改走 `StartDropEntry()`，不走傳統轉輪滾動，而是由 `DropEntryStrategy` / `CascadeFillStrategy` 控制符號從上方掉入
