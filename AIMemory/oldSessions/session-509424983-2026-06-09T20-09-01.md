幾種常用方式：

**1. GitHub Trending 頁面（最直覺）**

https://github.com/trending

可依語言、時間範圍（Today / This week / This month）篩選。

**2. GitHub Search API（程式化查詢）**

用 `gh` CLI 或直接打 API，按 stars 排序、限定建立時間：

```bash
# 最近 7 天建立、stars 最多的 repo
gh search repos --created=">2026-06-02" --sort=stars --order=desc --limit=20

# 指定語言
gh search repos --created=">2026-06-02" --language=TypeScript --sort=stars --limit=20
```

**3. GitHub Search 網頁進階語法**

在 github.com/search 搜尋框輸入：

```
stars:>100 created:>2026-06-01 sort:stars
```

可加 `language:Python`、`topic:ai-agent` 等條件。

**4. 第三方工具**

- **star-history.com** — 視覺化 star 成長曲線
- **ossinsight.io** — 即時 trending 分析、可看特定領域
- **GitHunt**（Chrome extension）— 每日推薦
