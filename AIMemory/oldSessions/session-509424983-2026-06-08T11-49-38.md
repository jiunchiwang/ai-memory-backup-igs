## GitHubTool 概覽

**用途**：基於 Streamlit 的 GitHub 組織管理 Web UI 工具，版本 1.3.0。

**技術棧**：Python 3.10+ / Streamlit / pandas / requests / keyring

**核心功能**：

| 頁面 | 功能 |
|------|------|
| 設定管理 | 新增/管理 GitHub 組織設定，Token 用 OS keyring 安全儲存 |
| 倉庫查詢 | 查詢組織倉庫清單、Custom Properties、團隊權限 |
| 批量建倉 | 從 Excel/CSV/TXT/貼上文字批量建立私有倉庫（含 Custom Properties 設定） |
| 批量權限 | 批量加入團隊或協作者到倉庫（支援混合模式 + 文字解析） |
| 團隊成員管理 | 批量新增/移除團隊成員 |
| 操作日誌 | 歷史操作紀錄 |

**架構**：

```
app.py                     # Streamlit 入口
pages/                     # 8 個 Streamlit 多頁面
services/
  config_manager.py        # 設定 CRUD + keyring token 儲存
  data_loader.py           # Excel/CSV/TXT 解析、欄位驗證、重複檢查
  github_service.py        # GitHub REST API 封裝（分頁、rate limit、驗證）
  operation_logger.py      # 操作日誌持久化
GithubAPI.py               # 早期 API 函式（建倉、加團隊/協作者、submodule 操作）
Github.py                  # 早期 CLI 腳本（讀 Excel → 批量加團隊）
data/config.json           # 組織設定持久化
data/operations.json       # 操作日誌
tests/                     # pytest + hypothesis 測試
```

**特點**：
- 組織 `IGS-ARCADE-DIVISION-RD2` 為主要操作對象
- 支援「【GitHub創建新倉庫】」、「【GitHub倉庫人員調整】」、「【GitHub團隊成員調整】」等固定格式的文字解析
- 多種輸入方式：Excel/CSV/TXT 上傳 + 直接貼上文字 + 手動輸入
- Token 不存明文，走 OS Credential Store
- `start.bat` / `start_hidden.vbs` 支援背景啟動 Streamlit server

**啟動方式**：`streamlit run app.py` 或直接跑 `start.bat`
